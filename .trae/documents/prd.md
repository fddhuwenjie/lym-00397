## 1. 产品概述

基于多层 Perlin/Simplex 噪声的 3D 程序化地形生成器，支持参数化控制、LOD 分级渲染、分层着色及 heightmap/OBJ 导出。面向游戏开发者、3D 艺术家和地形研究人员，提供字节级可复现的确定性地形输出。

- 核心价值：零后端依赖的纯前端地形引擎，同种子同参数输出严格一致，可导出标准格式供 DCC 工具使用
- 目标用户：独立游戏开发者、3D 场景艺术家、程序化生成研究者

## 2. 核心功能

### 2.1 功能模块

1. **3D 地形视口**：Three.js 渲染的交互式 3D 场景，支持轨道控制（旋转/缩放/平移）
2. **参数控制面板**：种子/振幅/频率/八度/lacunarity/persistence 等噪声参数实时调节
3. **LOD 调试视图**：可视化 chunk 边界与 LOD 级别，验证接缝无缝性
4. **导出面板**：16 位灰度 PNG heightmap + OBJ 模型导出

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 地形视口 | 3D 渲染区 | Three.js 渲染地形网格，OrbitControls 交互 |
| 地形视口 | 着色显示 | 根据高度+斜率分层着色：水/沙/草/岩/雪 |
| 地形视口 | LOD 切换 | 按相机距离自动切换 chunk 分辨率（4级） |
| 控制面板 | 噪声参数 | 种子、振幅、频率、八度、lacunarity、persistence 滑块 |
| 控制面板 | 地形参数 | 高度缩放、水面高度、chunk 尺寸、世界范围 |
| 控制面板 | 导出按钮 | 导出 PNG（16位灰度）、导出 OBJ |
| 控制面板 | LOD 调试 | 显示/隐藏 LOD 级别颜色叠加、chunk 边界线 |

## 3. 核心流程

用户打开页面 → 默认参数生成地形 → 3D 视口渲染 → 用户调节参数 → Web Worker 重新计算 heightmap → 主线程更新网格 → 用户导出 PNG/OBJ

```mermaid
flowchart TD
    "用户调节参数" --> "参数序列化"
    "参数序列化" --> "发送至 Web Worker"
    "发送至 Web Worker" --> "Worker: PRNG 初始化"
    "Worker: PRNG 初始化" --> "Worker: 多层噪声计算 heightmap"
    "Worker: 多层噪声计算 heightmap" --> "Worker: 分片处理大尺寸 heightmap"
    "Worker: 分片处理大尺寸 heightmap" --> "Worker: 网格生成 + 法线计算"
    "Worker: 网格生成 + 法线计算" --> "Worker: LOD 顶点/索引数据"
    "Worker: LOD 顶点/索引数据" --> "主线程: mesh upload"
    "主线程: mesh upload" --> "Three.js 渲染"
    "Three.js 渲染" --> "用户查看/导出"
    "用户查看/导出" --> "导出 PNG / OBJ"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色科技风（#0a0f1a 背景 + #00e5ff 青色高亮 + #ff6b35 橙色强调）
- **字体**：JetBrains Mono（数据/参数显示）+ Space Grotesk（UI 文字）
- **布局**：左侧 3D 视口全屏、右侧半透明控制面板浮层
- **风格**：工业/技术仪表盘风格，暗色调 + 发光边框 + 网格纹理

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 地形视口 | 3D 渲染区 | 全屏 Canvas，暗色天空盒，方向光+环境光 |
| 地形视口 | HUD 信息 | 左上角 FPS、三角面数、chunk 数量 |
| 控制面板 | 参数区 | 半透明毛玻璃面板，滑块组，数值输入框 |
| 控制面板 | 导出区 | 带图标按钮，导出进度条 |
| 控制面板 | LOD 调试 | 复选框：显示 LOD 级别色、显示 chunk 边界 |

### 4.3 响应式

- 桌面优先：3D 视口占主区域，控制面板右侧浮层
- 移动端适配：控制面板底部抽屉式展开

### 4.4 3D 场景指导

- 环境：深蓝色天空渐变，远处雾化效果
- 光照：定向光模拟太阳光（45° 角）+ 低强度环境光
- 相机：透视相机，初始位置俯视 45°，OrbitControls
- 后处理：可选雾化效果增强深度感
- 性能预算：保持 60fps，chunk 按需加载卸载
